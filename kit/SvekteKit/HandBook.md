# SvelteKit

## 创建项目

``` bash
npm create svelte project-name
```

## 文件结构

``` yaml
my-project/
├ src/
│ ├ lib/
│ │ └ [your lib files]  
│ ├ params/
│ │ └ [your param matchers]
│ ├ routes/
│ │ └ [your routes] 
│ ├ app.html
│ ├ error.html
│ └ hooks.js
├ static/
│ └ [your static assets]
├ tests/
│ └ [your tests]
├ package.json
├ svelte.config.js
├ tsconfig.json
└ vite.config.js
```

>特别注意`hooks.js`文件 ,`routes目录`,`params目录`
